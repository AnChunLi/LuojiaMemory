##  若比邻      
#### ――基于地理位置的社交应用

### 介绍及初心
这是一个基于定位的社交应用，通过地图为载体，用户可以发布自己动态、心情等，可以上传图片、上传视频、发布需求、寻找伙伴、分享音乐，甚至可以交友聊天，甚至可以做更多的事...总之是要做一款很酷、很有创意的互联网产品。


### 功能
#### ・		用户发布动态，包括文字。表情、图片（支持多张）。
#### ・		高德地图为载体，以火苗的形式在地图上展现用户动态，点击可查看。
#### ・		动态包括内容、图片、时间、地点、距离、热度等信息。用户可以点赞。
#### ・		动态按照热度排列为列表，可以切换至列表视图，可以按距离进行动态筛选。
#### ・		地图主题变换，可切换多种主题地图。

### 技术栈
接下来才是重点
前端是Vue.js开发，后端是原生PHP7.2，数据库MySQL5.7，地图借助高德地图API。
由于是小项目，就采用了PHP，之后可以采用PHP和node.js互搭的方式。因为我对他的定位一直是业余小项目，所以php快速开发够用了，暂不考虑用Java.
